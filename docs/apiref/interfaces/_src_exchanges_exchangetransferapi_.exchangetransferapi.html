requestTransfer

  • requestTransfer(request: TransferRequest): Promise<TransferResult>
    • Defined in src/exchanges/ExchangeTransferAPI.ts:34

    Request a transfer to another wallet or account on the same exchange. This operation should resolve near-instantaneously. If the exchange does not support this operation, the promise is rejected, but there are ways of implementing pseudo-wallets so that this operation can be emulated on exchanges that don't formally support multiple wallets.
