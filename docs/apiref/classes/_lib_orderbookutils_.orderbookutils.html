calculateMarketOrderStats

  • calculateMarketOrderStats(side: string, amount: Biglike, fees?: BigJS): MarketOrderStats
    • Defined in lib/OrderbookUtils.ts:138

    Calculate stats for a market order. If a cached version is available, it will use that, which is much more efficient if multiple calculations on the same book are required. Otherwise for small, once-off calculations it's better to use the naive approach

    Parameters

    • side: string

      Must be 'buy' or 'sell'

    • amount: Biglike

      The size of the trade

    • Default value fees: BigJS = ZERO

      [] Optional. The fee rate charged (as a fraction, NOT a percentage)

    Returns MarketOrderStats
