Class ExchangeRateFilter

Implements a stream filter that applies an exchange rate to a stream of Coinbase Pro messages.

If the FXService has any problems and enters an ErrorState, the filter will continue to convert prices, but you can watch for FXRateMayBeInvalid events and respond accordingly
