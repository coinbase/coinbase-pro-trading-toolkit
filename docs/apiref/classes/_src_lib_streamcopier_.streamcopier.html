Class StreamCopier

Clones a (readable) stream to multiple (writable) streams. Manages backpressure for each output stream, though if one stream is much slower than the others you could get large memory consumption. There's no way around this, except stopping the feed to all the other streams too.

Accepts an optional filter function that determines whether a message goes to any particular output stream (aka a devil's trapdoor).
