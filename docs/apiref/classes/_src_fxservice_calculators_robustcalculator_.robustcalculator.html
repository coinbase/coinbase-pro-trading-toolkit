Class RobustCalculator

Calculates exchange rates in a robust manner. The calculator queries exchange rates from multiple sources and then returns the average price for all the reliable sources.

A source is deemed reliable if it is less than a given distance from the median exchange rate AND the change is price is less than a given distance from the median change in price.

This should protect from erroneous rates being delivered in these cases:

  • one or more source is down, or providing faulty data for an indefinite period of time

  • A flash crash occurs on a source, providing poor data for a short amount of time.

    To be reliable, one should supply at least 5 rate sources for this method.
