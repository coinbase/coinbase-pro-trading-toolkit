FeedFactory

  • FeedFactory(logger: Logger, productIds: string[], auth?: ExchangeAuthConfig): Promise<BittrexFeed>
    • Defined in src/factories/bittrexFactories.ts:62

    This is a straightforward wrapper around getSubscribedFeeds using the Factory pattern with the most commonly used defaults. For customised feeds, use getSubscribedFeeds instead. It's really not adding much, but we keep it here to maintain a consistent method naming strategy amongst all the exchanges

    It is assumed that your API keys are stored in the BITTREX_KEY and BITTREX_SECRET envars
