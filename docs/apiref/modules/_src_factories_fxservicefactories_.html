SimpleFXServiceFactory

  • SimpleFXServiceFactory(provider?: string, logger?: Logger, refreshInterval?: number): FXService
    • Defined in src/factories/fxServiceFactories.ts:62

    Generate an return an FXService provider with sane defaults. If no arguments are specified, Yahoo Finance is used as the sole provider using a SimpleRateCalculator instance.

    The returned FXService has a default refresh interval of 10 minutes. By default, no currency pairs are set, so a recommended pattern is to set them directly after receiving the FXService, i.e.

      const service = SimpleFXServiceFactory().addCurrencyePair({ from: 'USD', to: 'EUR'});

    Parameters

    • Default value provider: string = "yahoo"

      Either 'yahoo', 'openexchangerates' or 'coinmarketcap'. For OER, the OPENEXCHANGE_KEY envar must be set

    • Optional logger: Logger

      If not specified a new ConsoleLogger will be created

    • Optional refreshInterval: number

      the period (in ms) to poll the underlying API for new prices

    Returns FXService
